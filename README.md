# Spa Service API
## About
Spa Service Rest API by Golang includes create, delete, view methods.
 
## Install
with [homebrew](http://mxcl.github.io/homebrew/):

```
brew install go
```

or

[install Golang manually](https://golang.org/doc/install)

## Gorrilla Mux

```
go get github.com/gorilla/mux
```

## Run


```
go run main.go
```
